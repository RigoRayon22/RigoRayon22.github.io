What is an Operating System?

An operating system (OS) is the software layer that helps your programs use hardware safely and efficiently. It manages key resources like the CPU, memory, and storage, and provides services such as files, networking, and security permissions.

  • Virtualization: makes limited hardware feel abundant (CPU, memory).
  • Abstraction: hides low-level details behind friendly interfaces.
  • Protection: stops programs from harming each other or the system.

CPU & Processes

The OS shares the CPU by running many programs “as if” they are at the same time. It uses scheduling and context switching to move between tasks.

Quick Glossary

Click to expand
  • Kernel: the privileged core of the OS.
  • System call: a controlled request for OS services.
  • Interrupt: a signal that makes the CPU stop and run OS code.
  • Virtual memory: the illusion of large, private memory per program.
