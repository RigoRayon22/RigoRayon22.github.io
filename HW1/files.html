What a File System Does

A file system organizes data on storage devices (SSD/HDD) into files and folders. It also manages metadata like names, permissions, sizes, and modification times.

Key Ideas

  • Directories: map names to files.
  • Blocks: fixed-size chunks used to store data on disk.
  • Metadata: info about a file (owner, size, timestamps).
  • Reliability: strategies to recover after crashes.

Many systems use techniques like journaling or copy-on-write to reduce corruption risks.

Why “crash consistency” matters

Power loss or sudden shutdown can interrupt writes. A well-designed file system tries to ensure you don’t end up with half-updated metadata or missing links in directories.
